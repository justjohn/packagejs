Modular JavaScript

packagejs provides a utility for packaging (memoizing) multiple CommonJS Modules/2.0 modules into a singe JavaScript file.

Usage

Usage:
    packagejs [--help] [--compress] [--output file] input.js

Options
    --help          Display this help message.
    --compress      Use UglifyJS to compress the output file.
    --output [file] The file name for the output of packagejs.

See the demos/ directory for runnable examples of the functionality.
